## 2231. 분해합
```python
N = int(input())
for i in range(1,N+1) :
    sum_ = sum(map(int,str(i))) # i를 str 처리 하여 각 자리수의 합을 구함
    total_sum = sum_ + i # 분해합 = 자기자신 + 각 자리수의 합
    if total_sum == N :
        print(i)
        break
    if i == N : # 입력값과 생성자가 같다 = 생성자가 없다.
        print(0)
```
