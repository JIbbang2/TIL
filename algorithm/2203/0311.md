## 2884. 알람 시계
```python
H,M = map(int,input().split())
M -= 45
if M < 0 :       # 분이 음수가 되면
    H -= 1       
    M += 60
    if H < 0 :   # 시가 음수가 되면
        H += 24

print(H, M)
```





## 2908. 상수

```python
A,B = map(str, input().split())

nA = A[::-1]
nB = B[::-1]
print(max(nA,nB))
```





## 3052. 나머지

```python
lst = []                 # 나머지들을 담아둘 리스트
for k in range(10):
    N = int(input())
    lst.append(N % 42)

li = [0]*43              # 빈 리스트를 만든 뒤 (나머지가 41까지 있으니까 넉넉히)
for i in lst:            # lst의 요소를 인덱스로 해서 1씩 증가시키기
    li[i] += 1

total = 0
for j in li : 
    if j != 0 :          # 0이 아니다 = 값이 있다 -> 세어주기
        total += 1

print(total)
```
