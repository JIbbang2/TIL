## 8958. OX퀴즈
```python
T = int(input())
for i in range(T):         # 테스트케이스만큼 반복하여 인풋 받기
    user_input = input()

    total = 0              # 전체 합을 저장할 total 과
    score = 0              # 연속된 점수를 셀 score 초기화
    for j in user_input:
        if j == 'O' :
            score += 1
            total += score
        elif j == 'X':    # X일때는 score을 초기화해서 다시 1부터 세도록 만들기
            score = 0

    print(total)
```
