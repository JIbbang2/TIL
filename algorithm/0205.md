# 1. map, split 복습

```python
# 두 수를 공백 기준으로 입력받을 때
a, b = map(int, input().split())

# map(x, iterable) : iterable을 x형식으로 변환하여 반환
# split([y]) : y를 단위로 나누어 리스트 반환
```

# 2. 소수점 표현하기
``` python
# round함수 사용. rount(값, 자릿수) -> 입력한 자릿수에서 값을 반올림
a = 4.27552
print(round(a, 3)) # 4.276

# f-string 사용. f'{값:.nf} -> 값을 n의자리에서 반올림
a = 4.27552
print(f'{a : .2f}') # 4.28
