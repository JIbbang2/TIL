## 10950. A+B - 3

```python
for tc in range(T):
    A, B = map(int,input().split())
    print(A+B))
```





## 10951. A+B - 4

```python
while True:
    try :
        A, B = map(int,input().split())
        print(A+B)
    except :
        break
```

> 입력이 없을때까지 출력 해본적이 없어서 해당 방법에 대해 공부함.
>
> 우선 while문으로 입력을 계속 받고
>
> try / except 문으로 오류가 발생되면 반복문을 종료하기.





## 10952. A+B - 5

```python
while True:
    A, B = map(int,input().split())
    if A == 0 and B == 0 :
        break
    else :
        print(A+B)
```
