## 10809. 알파벳 찾기
```python
s = input()
lst = 'abcdefghijklmnopqrstuvwxyz'
my_li = []
for i in lst :
    my_li.append(s.find(i))
print(*my_li)
```

> find 메서드 사용하기. index를 사용하면 오류가 뜸
