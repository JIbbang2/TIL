##### 1. 버블 정렬
> 인접한 두 개의 원소를 비교하며 자리를 교환하는 방식
ex) 최댓값, 최솟값 구하기

- 시간 복잡도 : O(n**2)

```python
# 최댓값 구하기
li = [1,7,4,3,6]

# 리스트 맨 첫번째 값을 최댓값을 의미하는 변수에 할당
max_v = li[0] 
for i in range(1, len(li)-1) :
    if max_v < li[i] :
        max_v = li[i]
    print(max_v)
```
